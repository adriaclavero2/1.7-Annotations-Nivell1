Description

Learning project in Java to explore the behavior of annotations such as @Override and @Deprecated. The goal is to understand how annotations enrich the code by providing metadata that affects compilation and execution, and how they help enforce good practices or mark obsolete methods.

From the main() method in the Main class, invoke the methods to demonstrate the effect of @Override.
